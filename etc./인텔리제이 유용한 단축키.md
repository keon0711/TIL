# 인텔리제이 유용한 단축키

| 단축키   | 기능                   |
| ----- | -------------------- |
| ⌥ 'x' | 사용하지 않는 모든 탭 닫기      |
| ⇧⌘V   | 붙여넣을 콘텐츠 선택          |
| ⌥/    | Hippie 코드 완성 사용      |
| ⌥⌘L   | 코드 서식 다시 지정          |
| ⌘F12  | 파일 구조 탐색             |
| ⌃G    | 찾은 항목 여러 개 선택        |
| ⌘⇧↑/↓ | 줄 이동                 |
| ⌘D    | 줄 복제                 |
| ⌥ F3  | 연상 기호 북마크 추가         |
| ^`    | 뷰 모드 변경              |
| ⌘L    | 줄:열로 이동              |
| ⌘-/+  | 코드 블록 접기 및 펼치기       |
| ⌘⇧A   | 액션 찾기 (예: 배경 이미지 추가) |
