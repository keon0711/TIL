## 인증 VS 인가
### 인증 (Authenfication)
- 사용자 본인이 맞는지 확인

### 인가 (Authorization)
- 리소스에 접근 권한이 있는지 확인하는 

## OAuth
- **Access Code는 사용자(Resource Owner)가 알아야 할 필요가 없다. (브라우저에 저장 금지)**
- 로그인만 할거면 Access Tocken이 필요하지 않음
- Access Tocken은 Client가 Resource에 접근할 때 사용
